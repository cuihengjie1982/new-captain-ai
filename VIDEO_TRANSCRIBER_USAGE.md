# 视频逐字稿生成器使用说明

## 功能概述

视频逐字稿生成器使用阿里云通义听悟技术，可以将视频文件转换为带时间戳的精确逐字稿。支持多种导出格式，包括纯文本、SRT字幕文件和VTT字幕文件。

## 功能特性

### 1. 智能转写
- 支持多种视频格式（MP4, AVI, MOV, WMV）
- 自动识别语音内容并生成精确转写
- 支持中英文混合识别
- 智能标点预测和格式化

### 2. 时间戳精度
- 毫秒级精确时间戳
- 逐句分割，便于定位
- 支持说话人识别（如适用）

### 3. 多格式导出
- **纯文本**：连续的文字内容
- **SRT字幕**：适用于视频播放器
- **VTT字幕**：Web视频标准格式

### 4. 交互式播放
- 集成视频播放器
- 点击文字段落可跳转到对应时间点
- 实时高亮当前播放段落

## 使用步骤

### 1. 上传视频文件

1. 在解决方案页面点击"视频逐字稿"标签
2. 点击上传区域或拖拽视频文件
3. 支持最大2GB的视频文件
4. 确保视频包含清晰的语音内容

### 2. 开始转写

1. 选择视频文件后，点击"开始转写视频"按钮
2. 系统会自动：
   - 上传视频到阿里云服务
   - 提交转写任务
   - 处理并生成逐字稿
3. 转写过程通常需要几分钟，具体取决于视频长度

### 3. 查看结果

转写完成后，您可以：

- **查看逐字稿**：按时间顺序排列的完整文字内容
- **播放视频**：使用集成播放器观看视频
- **交互导航**：点击文字段落跳转到对应时间点
- **内容摘要**：查看AI生成的内容摘要（如适用）

### 4. 导出下载

提供多种导出选项：

- **下载文本稿**：纯文本格式，便于阅读和编辑
- **下载SRT字幕**：用于视频播放器
- **下载VTT字幕**：用于Web视频
- **复制文本**：直接复制到剪贴板

## 技术支持

### 环境配置

确保以下环境变量已正确配置：

```env
# 阿里云通义听悟配置
VITE_TINGWU_APPKEY=your_tingwu_appkey
VITE_TINGWU_TOKEN=your_tingwu_token

# 阿里云千问API配置
VITE_QWEN_API_KEY=your_qwen_api_key
```

### API配置说明

- **VITE_TINGWU_APPKEY**：通义听悟应用密钥
- **VITE_TINGWU_TOKEN**：访问令牌，需要定期更新
- **VITE_QWEN_API_KEY**：通义千问API密钥（用于智能分析）

## 常见问题

### 1. 转写失败
- 检查网络连接
- 确认API配置正确
- 验证视频文件格式和大小

### 2. 精度问题
- 确保音频质量清晰
- 避免背景噪音干扰
- 使用标准发音和语速

### 3. 文件大小限制
- 当前支持最大2GB的视频文件
- 大文件建议先压缩处理

### 4. 语言支持
- 主要支持中文识别
- 英文混合识别效果良好
- 其他语言支持有限

## 注意事项

1. **隐私保护**：请勿上传包含敏感信息的视频内容
2. **版权合规**：确保有视频内容的处理权限
3. **费用控制**：大量使用可能产生API费用
4. **质量要求**：音频质量直接影响转写效果
